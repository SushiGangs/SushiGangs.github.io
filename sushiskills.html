SushiSkills

Core skill engine for Sushi Ecosystem - Tạo và quản lý kỹ năng chiến đấu đỉnh cao.

v1.0.0 Yêu cầu: SushiLib, SushiCore

Mô Tả

Tải Xuống Ngay

1. Tổng Quan Kỹ Năng (Skill Engine)

SushiSkills là một skill engine hoàn chỉnh được thiết kế đặc biệt cho hệ sinh thái Sushi MMORPG. Hỗ trợ hệ thống combat đa dạng:

  • ⚔️ Damage System với Stat Scaling và Critical hits (Chí mạng).
  • 💚 Healing System dành cho các class Hỗ trợ (Healers).
  • 🔵 Quản lý tiêu hao Mana / Stamina.
  • ✨ Hệ thống Aura & Buff (Trạng thái và sát thương theo thời gian).
  • 🎯 Advanced Targeting: Hỗ trợ target theo bán kính, nón (cone), đường thẳng.

2. Cú Pháp Công Thức (Expression Syntax)

Bạn có thể sử dụng biểu thức toán học trực tiếp trong cấu hình kỹ năng:

// Tính sát thương dựa trên chỉ số người chơi
"50 + @STRENGTH * 2.5"
"clamp(50 + @INT * 3, 10, 200)"
"@skill:combo >= 3 ? 100 : 50"

3. Cấu hình Kỹ năng (Configuration)

Mỗi kỹ năng được định nghĩa rõ ràng với conditions (điều kiện) và mechanics (chuỗi hành động):

skills:
  fireball:
    name: "§c§lFireball"
    icon: FIRE_CHARGE
    cooldown: 5.0
    conditions:
      - "mana{cost=30}"
      - "talent{node=mage_fire}"
    mechanics:
      - "damage{amount=60+@INT*3.5;element=FIRE;target=@target}"
      - "dot{damage=10;interval=1s;duration=5s;type=FIRE;target=@target}"
      - "particle{type=FLAME;count=20;target=@target}"

4. Tích Hợp Hệ Sinh Thái (Integration)

SushiSkills gọi trực tiếp các API của hệ sinh thái:

  • SushiLib: Sử dụng ResourceAPI để trừ Mana, StatAPI để lấy thuộc tính, AuraManager để gắn hiệu ứng đốt cháy.
  • SushiCore: Kiểm tra người chơi đã mở khóa kỹ năng đó trên Cây Kỹ Năng (Talent Tree) hay chưa.

5. Tối ưu Hiệu Suất

  • Caching các công thức toán (Expression Engine).
  • Giới hạn số lượng Particle hiển thị.
  • Sử dụng khoảng cách bình phương (Distance-squared) để tính toán vùng AOE siêu mượt.
