SushiLib Skills System

Hệ thống kỹ năng RPG mạnh mẽ - Tạo skills phức tạp và thú vị chỉ bằng file YAML đơn giản!

Minecraft:1.16+
Java:Java 8+
Dependencies:SushiLib, Vault

Skills Admin Guide

🌟 Tổng Quan

SushiLib Skills System là plugin kỹ năng RPG cho server Minecraft. Plugin cho phép bạn tạo skills phức tạp và thú vị mà không cần biết code, chỉ cần chỉnh sửa file YAML đơn giản!

📦 Cài Đặt Plugin

1. Yêu Cầu Hệ Thống

  • Minecraft Version: 1.16+
  • Java Version: Java 8+
  • Dependencies: Vault (bắt buộc), WorldGuard (tùy chọn)

2. Cài Đặt

  1. Tải file SushiLib.jar vào thư mục plugins/
  2. Khởi động lại server
  3. Plugin tự động tạo thư mục và file config

3. Cấu Trúc File

plugins/SushiLib/
├── config.yml              # Cấu hình chính
├── skills/                  # Thư mục chứa các skill
│   ├── combat_skills.yml   # Skills chiến đấu
│   ├── magic_skills.yml    # Skills phép thuật
│   └── utility_skills.yml  # Skills tiện ích
├── skilltrees/             # Cây kỹ năng
│   └── default_tree.yml    
└── data/                   # Dữ liệu người chơi
    └── players/

⚙️ Commands & Permissions

Admin Commands

# Quản lý skills
/sushilib reload skills              # Reload tất cả skills
/skill reload <skill_name>           # Reload skill cụ thể
/skill validate                      # Kiểm tra lỗi cấu hình
/skill give <player> <skill_name>    # Cho skill cho player
/skill reset <player>                # Reset skills của player

# Debug
/skill debug <player>                # Bật/tắt debug mode
/skill info <skill_name>             # Thông tin chi tiết skill
PermissionMô tả
sushilib.admin.*Tất cả quyền admin
sushilib.skill.reloadReload skills
sushilib.skill.giveCho skill cho người khác

Player Commands

/skills                    # Mở GUI skills
/cast <skill_name>         # Cast skill
/skilltree                 # Mở cây kỹ năng
/mana                     # Xem mana hiện tại
PermissionMô tả
sushilib.skill.useSử dụng skills
sushilib.skill.<skill_name>Permission skill cụ thể

📝 Tạo Skills Đầu Tiên

Bước 1: Tạo File Skill

Tạo file mới trong skills/my_skills.yml:

# Skills của tôi
fireball:
  display-name: "&c&lQuả Cầu Lửa"
  icon: "FIRE_CHARGE"
  cooldown: 8.0
  mana-cost: 30.0
  
  description:
    - "&7Bắn một quả cầu lửa"
    - "&7gây 40 sát thương"
  
  mechanics:
    - "sound{type=ENTITY_BLAZE_SHOOT; volume=1.0} @self"
    - "projectile{type=FIREBALL; velocity=2.0} @crosshair{range=25.0}"
    - "damage{amount=40.0; type=FIRE} @radius{r=3.0}"
    - "particle{type=EXPLOSION_LARGE; amount=3} @impact"

Bước 2: Reload Skills

/sushilib reload skills

Bước 3: Test Skill

/skill give <yourname> fireball
/cast fireball

🎯 Skill Configuration Basics

Thuộc Tính Cơ Bản

skill_name:
  display-name: "&cTên Hiển Thị"     # Tên skill (hỗ trợ color codes)
  icon: "FIRE_CHARGE"                 # Icon trong GUI
  cooldown: 5.0                       # Thời gian hồi chiêu (giây)
  mana-cost: 25.0                     # Mana cần thiết
  charges: 1                          # Số lần tích trữ (optional)
  
  description:                        # Mô tả skill
    - "&7Dòng mô tả 1"
    - "&7Dòng mô tả 2"
    
  permission: "my.skill.fireball"     # Permission cần thiết (optional)
  level-requirement: 10               # Level tối thiểu (optional)

Mechanics - Hiệu Ứng Skill

mechanics:
  - "damage{amount=30.0} @target"                    # Gây sát thương
  - "heal{amount=25.0} @self"                        # Hồi máu
  - "particle{type=FLAME; amount=20} @self"          # Hiệu ứng visual
  - "sound{type=ENTITY_EXPLODE; volume=2.0} @self"   # Âm thanh
  - "potion{type=STRENGTH; duration=200} @self"      # Buff effect

⚙️ Danh Sách Mechanics

SushiLib hỗ trợ đa dạng các loại Mechanic (hành động/hiệu ứng) khác nhau như Damage, Projectile, Particle, Potion, v.v. Nhấn vào nút bên dưới để xem danh sách đầy đủ chi tiết các thông số, cách thiết lập và ví dụ mẫu cấu hình cho từng loại:

Xem Tra Cứu Mechanics

🎯 Targeting System

@self                           # Bản thân
@crosshair{range=20.0}         # Mục tiêu đang nhắm
@radius{r=8.0}                 # Tất cả trong phạm vi 8 blocks
@radius{r=5.0; maxTargets=3}   # Tối đa 3 mục tiêu gần nhất
@line{r=15.0; w=2.0}           # Đường thẳng dài 15, rộng 2 blocks
@cone{range=12.0; angle=60.0}  # Hình nón 60 độ, tầm 12 blocks

⚡ Advanced Features

Multi-Hit Skills

triple_strike:
  display-name: "&6Triple Strike"
  mechanics:
    - "damage{amount=20.0} @crosshair{range=8.0}"
    - "delay{ticks=5} @self"              # Đợi 0.25 giây
    - "damage{amount=20.0} @crosshair{range=8.0}"
    - "delay{ticks=5} @self"
    - "damage{amount=20.0} @crosshair{range=8.0}"

Conditional Skills

execute:
  display-name: "&4Execute"
  mechanics:
    # Nếu mục tiêu có ít hơn 25% HP, gây 999 sát thương
    - "conditional{
        condition=target-health < 25%;
        trueActions=['damage{amount=999.0}'];
        falseActions=['damage{amount=30.0}']
      } @crosshair{range=10.0}"

Chain Skills

chain_lightning:
  display-name: "&bChain Lightning"  
  mechanics:
    - "chain{
        damage=25.0;
        maxTargets=5;
        jumpRange=8.0
      } @crosshair{range=20.0}"

🔧 Configuration Options

config.yml Settings

sushilib:
  skills:
    enabled: true
    default-mana: 100.0           # Mana mặc định
    mana-regen-rate: 2.0          # Hồi mana/giây
    cooldown-reduction: 0.0       # Giảm cooldown toàn bộ (%)
    
  gui:
    skills-title: "&9Your Skills"
    skill-tree-title: "&6Skill Tree"
    
  messages:
    insufficient-mana: "&cKhông đủ mana! Cần %required%, hiện có %current%"
    skill-on-cooldown: "&cSkill đang hồi chiêu! Còn %time% giây"
    skill-learned: "&aĐã học skill: &f%skill%"

🐛 Troubleshooting

1. Skill không load được

# Kiểm tra syntax YAML
/skill validate

# Check console logs
tail -f logs/latest.log | grep SushiLib

2. Skill không gây sát thương

# Đảm bảo có @target hoặc @radius
- "damage{amount=30.0} @crosshair{range=10.0}"  # ✅ Correct
- "damage{amount=30.0}"                         # ❌ Missing target

3. Particle không hiện

# Đảm bảo có target location
- "particle{type=FLAME; amount=20} @self"       # ✅ Correct
- "particle{type=FLAME; amount=20}"            # ❌ Missing location

4. Permission Issues

# Cấp permission đúng cách
/luckperms user <player> permission set sushilib.skill.use true
/luckperms user <player> permission set sushilib.skill.fireball true

💡 Tips & Best Practices

Performance Tips

  • Giới hạn particle amount (< 50 trong AOE skills)
  • Sử dụng delay hợp lý giữa các mechanics
  • Tránh targeting radius quá lớn (< 15 blocks)

Balance Tips

  • Skill mạnh = cooldown cao + mana cost cao
  • AOE skills nên có cast time
  • Single target skills có thể instant cast

User Experience

  • Luôn có âm thanh và visual feedback
  • Mô tả skill rõ ràng trong description
  • Icon phù hợp với theme của skill
