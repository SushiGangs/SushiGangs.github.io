SushiCore

Hệ thống RPG và Life Skills hoàn chỉnh cho Server của bạn.

Tải Xuống Ngay
Phiên bản:1.20+
Yêu cầu:SushiLib

Mô Tả

1. Tổng Quan Kiến Trúc

SushiCore la plugin quan ly trung tam cua he sinh thai, cung cap nen tang RPG vung chac. Duoc viet cho Paper 1.20.4+ va Java 21.

Cấu trúc Module bao gồm:

  • RPG System (Class, Level, XP)
  • Talent System (Skill Tree)
  • Combat System (Skills, Combat Mode)
  • Party System (Groups, Buffs)
  • Gathering System (Professions)
  • Thien Phu System (Gacha Traits)

2. Hệ Thống RPG Cốt Lõi

Quản lý Chức nghiệp (Class): YAML-based definitions. Hỗ trợ Base Stats, giới hạn vũ khí, đường cong kinh nghiệm (XP curves).

name: "Chiến Binh"
max_level: 100
xp_curve: "NORMAL"
skill_tree: "warrior_tree"

Dữ liệu người chơi (PlayerData) được lưu trữ an toàn bằng SushiLib metadata, auto-save mỗi 5 phút và khi ngắt kết nối.

3. Combat & Skill Engine

Hệ thống chiến đấu Virtual Hotbar độc quyền:

  • Bấm F (Swap Hand) để vào/ra Combat Mode.
  • Kích hoạt chiêu thức thông qua phím 1, 2, 3 trên Hotbar.
  • Khóa tương tác hòm đồ (Inventory Block) trong lúc Combat.

4. API Integration (Dev)

PlayerData data = SushiCore.getInstance().getPlayerManager().getData(player);
boolean hasTalent = data.hasTalent("fireball_mastery");

PartyProvider party = SushiCore.getInstance().getPartyManager();
if (party.isPartyMember(playerA, playerB)) {
    // Không gây sát thương đồng đội (Friendly Fire)
}

5. Lệnh & Quyền quản trị

  • /rpg class set <p> <class> - Cài đặt class
  • /rpg addxp <p> <amount> - Thêm điểm XP
  • /skillstree, /party, /thienphu - Các lệnh giao diện

Quyền quản trị chính yếu: sushicore.admin
