SushiLib v3.0

Thư viện lõi độc quyền cung cấp API mạnh mẽ, quản lý chỉ số RPG, sát thương, hệ nguyên tố và tối ưu hóa cơ sở dữ liệu.

Tải Xuống
Phiên bản:1.20+
Yêu cầu:PaperSpigot / Vault

Mô Tả

Tính Năng Nổi Bật

  • Hệ thống Chỉ số RPG (Stats API):
    • Quản lý máu, năng lượng, thể lực, tốc độ di chuyển, sát thương, chí mạng, đỡ đòn, v.v.
    • Hỗ trợ công thức tính toán tuỳ chỉnh hoàn toàn thông qua file stats.yml.
    • Hệ thống giới hạn chỉ số (min-max-values), quy định base-stat và định dạng số thập phân.
  • Hệ thống Nguyên Tố (Element System): Cộng hưởng sát thương hệ Lửa, Nước, Đất, Gió thông qua file elements.yml.
  • Hệ thống Combat (DamageManager): Tính toán sát thương tự động với xuyên giáp, kháng hệ, phòng thủ, lifesteal và phản đòn.
  • Hiệu ứng Tạm thời (Aura & Temp Stat): Thêm chỉ số hoặc hiệu ứng tạm thời trong thời gian nhất định.
  • Tối ưu Cơ Sở Dữ Liệu: Sử dụng SQLite local hoặc MySQL (HikariCP) để lưu trữ chỉ số. Đồng bộ đa luồng.
  • Tương thích PlaceholderAPI: Cung cấp hàng loạt biến số cho Holograms, Scoreboard.

Cầu Hinh (Configs)

Ví dụ cấu hình stats.yml:

stats:
  HEALTH:
    display_name: "&cMáu"
    base_value: 20.0
    max_value: 20000.0
  DAMAGE:
    display_name: "&cSát Thương"
    base_value: 1.0
    min_value: 0.0

Dành cho Developer (API)

// Lấy PlayerData
SushiPlayerData data = SushiLib.getAPI().getPlayerData(player.getUniqueId());
double health = data.getStatValue("HEALTH");

// Thêm chỉ số tạm thời (Temp Stat) kéo dài 60s
data.addTempStat("DAMAGE", 50.0, 60);

// Kích hoạt tính toán sát thương tuỳ chỉnh
SushiDamageEvent event = new SushiDamageEvent(attacker, victim, baseDamage);
Bukkit.getPluginManager().callEvent(event);
