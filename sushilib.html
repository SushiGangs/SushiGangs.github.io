Thư Viện Lõi SushiLib

Nền tảng API mạnh mẽ cung cấp hệ thống RPG hoàn chỉnh - Quản lý chỉ số, sát thương, hệ nguyên tố, và tối ưu hóa cơ sở dữ liệu.

PaperSpigot 1.20+ Java 21 SQLite / MySQL

Các Module Chính

Skills System

Hệ thống kỹ năng RPG mạnh mẽ. Tạo skill phức tạp chỉ bằng file YAML - hỗ trợ Projectile, AOE, Teleport, Conditional và nhiều hơn nữa.

Thông Tin Nhanh

Tính Năng Nổi Bật

  • Hệ thống Chỉ số RPG (Stats API): Quản lý máu, năng lượng, thể lực, tốc độ di chuyển, sát thương, chí mạng.
  • Hệ thống Nguyên Tố (Element System): Cộng hưởng sát thương hệ Lửa, Nước, Đất, Gió.
  • Hệ thống Combat (DamageManager): Tính toán sát thương tự động với xuyên giáp, kháng hệ, phòng thủ, lifesteal.
  • Hiệu ứng Tạm thời (Aura & Temp Stat): Thêm chỉ số hoặc hiệu ứng tạm thời trong thời gian nhất định.
  • Tối ưu Cơ Sở Dữ Liệu: Sử dụng SQLite local hoặc MySQL (HikariCP).
  • Tương thích PlaceholderAPI: Cung cấp hàng loạt biến số cho Holograms, Scoreboard.
